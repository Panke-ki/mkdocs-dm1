## Установка MkDocs

Если вы хотите создать аналогичный блог, вот несколько команд:

```bash
# Установите MkDocs и плагины
pip install -r requirements.txt

# Создайте новый проект
mkdocs new my-blog

# Запустите локальный сервер
mkdocs serve

# Соберите сайт
mkdocs build
```
